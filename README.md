# 📚 Loksewa Preparation Web App

> A modern, responsive web application designed to help students prepare for Nepal Loksewa examinations through organized practice questions, quizzes, and progress tracking.

## 🚀 Features

* 📖 1000+ Loksewa practice questions
* 🎯 Questions categorized from Beginner to Advanced
* 📝 Interactive quizzes with instant results
* 📊 Progress tracking
* 🔍 Search and filter questions
* 📱 Fully responsive design
* ⚡ Fast and lightweight
* 🌙 Clean and modern user interface
* 💾 Runs directly in the browser (no installation required)

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 Purpose

This project was created to make Loksewa exam preparation easier by providing a free and accessible platform where users can practice questions anytime and improve their performance.

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/sagar10-web/loksewa-web-app.git
```

2. Open the project folder.

3. Launch `index.html` in your browser or use **Live Server** in VS Code.
